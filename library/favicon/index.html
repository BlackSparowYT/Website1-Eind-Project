Favicon

Favicon HTML:
To make this work is nothing special, just add the link element with its tags. (make sure the image is an .png, .jpg, .svg, .gif or .ico)

<link rel="icon" type="image/x-icon" href="link.to-favicon.com">




No CSS needed for a Favicon!
