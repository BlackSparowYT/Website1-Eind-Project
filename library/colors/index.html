Links CSS:
Make sure to add the "color: [color-name];" to the CSS.

#text1  {
  /*No CSS needed*/
}

#text2  {
  color :  #FF0000 ;   /*Makes the text red*/
}

#text3  {
  color :  #FF0000 ;   /*Makes the text red*/
  text-decoration :  line-through ;   /*Adds a line through the text, this also gets the same color as the text*/
}
